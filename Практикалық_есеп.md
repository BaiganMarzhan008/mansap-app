# Практикалық жұмыс бойынша қорытынды есеп

**1. Жоба атауы:**  
«Mansap App» — Жұмыс іздеушілер мен жұмыс берушілерге арналған платформа.

**2. Қолданылған технологиялар:**  
- **Frontend:** React, Vite, CSS (Responsive Design), PWA (Progressive Web App).  
- **Backend:** Node.js, Express.js.  
- **Database ORM:** Sequelize.  
- **Қосымша интеграциялар:** Socket.io (Нақты уақыттағы чат үшін), Google Gemini AI (Жасанды интеллект көмекшісі үшін).

**3. Сервер атауы (Хостинг):**  
- **Frontend:** Vercel  
- **Backend & Database:** Render.com

**4. Жоба сілтемесі:**  
[https://mansap-app.vercel.app/](https://mansap-app.vercel.app/)

**5. Git репозиторий сілтемесі:**  
[https://github.com/BaiganMarzhan008/mansap-app.git](https://github.com/BaiganMarzhan008/mansap-app.git)

**6. Мәліметтер базасы түрі:**  
PostgreSQL (Реляциялық мәліметтер базасы).

**7. Орындалған функционалдар тізімі:**  
- **Аутентификация және Авторизация:** Қолданушыларды тіркеу, жүйеге кіру, құпия сөзді қалпына келтіру (JWT токендер арқылы).  
- **Рөлдік басқару:** Жұмыс іздеуші (USER), Жұмыс беруші (EMPLOYER) және Әкімші (ADMIN) рөлдері.  
- **CRUD операциялары:** Вакансияларды қосу, өңдеу, өшіру және оқу. Резюме жасау.  
- **Іздеу және Фильтрлеу:** Жұмыстарды категориялары мен кілт сөздер арқылы іздеу.  
- **Өтінім қалдыру:** Вакансияларға өтінім (application) жіберу және олардың статусын басқару.  
- **Чат жүйесі:** Жұмыс беруші мен іздеуші арасындағы тікелей хабарлама алмасу (WebSocket).  
- **AI Ассистент:** Мансаптық кеңестер беретін және түйіндеме жазуға көмектесетін жасанды интеллект.  
- **PWA (Progressive Web App):** Сайтты ұялы телефонға және компьютерге қосымша (приложение) ретінде орнату мүмкіндігі.

**9. Тестілеу нәтижесі:**  
- **Функционалдық тестілеу:** Тіркелу, кіру және жұмыс қосу процестері 100% қатесіз жұмыс істеді.  
- **API тестілеу:** Барлық REST API endpoint-тері (200, 201, 400, 404, 500 status code) тексерілді.  
- **Мобильді тестілеу:** Сайттың адаптивті (responsive) дизайны телефон браузерлерінде дұрыс ашылды.  
- **PWA тестілеу:** Chrome және Safari браузерлері арқылы «Add to Home Screen» функциясы сәтті іске асты.

**10. Кездескен қателер және шешу жолдары:**  
1. *Қате:* Жергілікті (локальный) PostgreSQL базасына қосылу кезінде «The server does not support SSL connections» қатесі шықты.  
   *Шешімі:* `backend/config/database.js` файлындағы `dialectOptions.ssl` баптауын алып тастау арқылы шешілді.  
2. *Қате:* Терминалдан GitHub-қа жүктеу кезінде `fatal: not a git repository` қатесі.  
   *Шешімі:* Команда қате папкада орындалғаны анықталып, `cd JobPortal` арқылы негізгі папкаға өтіп, ескі `origin` өшіріліп, жаңа `git remote add origin` жасалды.  
3. *Қате:* Сайтты PWA ретінде орнату батырмасы шықпады.  
   *Шешімі:* `vite.config.js` ішіндегі PWA баптауларында иконкаларға `purpose: "any maskable"` қасиеті қосылды, содан кейін браузер иконканы сәтті қабылдап, орнатуға рұқсат берді.

**11. Қорытынды:**  
Практикалық жұмыс барысында локальді ортада жасалған «Mansap App» веб-қосымшасы толығымен бұлтты серверлерге (Vercel және Render) көшірілді. Жоба өзінің мәліметтер базасымен байланысып, интернетте нақты жұмыс істейтін өнімге (Production) айналды. Сонымен қатар, қауіпсіздік шаралары (құпия кілттерді `.env` ішіне жасыру, CORS) сақталып, заманауи PWA технологиясы енгізілді. Жоба барлық қойылған талаптарға толық жауап береді.
