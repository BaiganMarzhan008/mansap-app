# Mansap App (Job Portal)

Бұл жоба жұмыс іздеушілер мен жұмыс берушілерге арналған платформа.

## Орнату және іске қосу

1. **Репозиторийді жүктеп алу:**
   ```bash
   git clone <сіздің-репозиторий-сілтемеңіз>
   cd JobPortal
   ```

2. **Бэкендті іске қосу:**
   ```bash
   cd backend
   npm install
   ```
   `.env.example` файлын көшіріп, атын `.env` деп өзгертіңіз және ішіне өз мәліметтеріңізді жазыңыз.
   ```bash
   node server.js
   ```

3. **Фронтендті іске қосу:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Серверге орналастыру (Deploy)
Бұл жоба Render (Бэкенд) және Vercel (Фронтенд) үшін оңтайландырылған. Толық нұсқаулықты Практикалық жұмыс №6 бойынша орындауға болады.
