КІРІСПЕ

Қазіргі заманда ақпараттық технологиялардың қарқынды дамуы еңбек нарығына да айтарлықтай өзгерістер енгізуде. Жұмыс іздеу, қызметкерлерді іріктеу және түйіндеме ұсыну процестері біртіндеп цифрлық форматқа көшуде. Осы мәселелерді шешу үшін заманауи веб-платформаларды әзірлеу маңызды болып табылады.
Жобаның мақсаты – жұмыс іздеушілер мен жұмыс берушілерді байланыстыратын ыңғайлы, сенімді және функционалды веб-платформа әзірлеу.
Зерттеу объектісі – жұмыс іздеу және түйіндеме жариялау веб-платформасы. Жоба Frontend бөлігі Vercel платформасында, ал Backend бөлігі Render сервисінде орналастырылды. Курстық жұмыстың нәтижесінде әзірленген «Mansap App» веб-платформасы жұмыс іздеу процесін оңтайландырып, пайдаланушыларға ыңғайлы орта ұсынады.

---

1. БАҒДАРЛАМАЛЫҚ ӨНІМДІ ЖОБАЛАУ ЖӘНЕ АРХИТЕКТУРАСЫ

1.1 Жобаның мақсаты және технологиялық стек
Жоба атауы: «Mansap App» веб-платформасы.
Бұл жоба еңбек нарығындағы бюрократиялық кедергілерді азайтуға, жұмыс іздеу және түйіндеме жариялау процесін цифрландыруға бағытталған.
Қолданылған технологиялар:
- Frontend: React, Vite, CSS (Responsive Design), PWA (Progressive Web App).
- Backend: Node.js, Express.js.
- Қосымша интеграциялар: Socket.io (Чат), Google Gemini AI (Жасанды интеллект).

1.2 Деректер қоры және жүйе архитектурасы
Мәліметтер базасы түрі: PostgreSQL (Реляциялық мәліметтер базасы). Деректермен жұмыс істеу үшін Sequelize ORM қолданылды.
Жүйенің архитектурасы классикалық клиент-серверлік модельге негізделген. Пайдаланушы веб-браузер немесе мобильді құрылғы арқылы жүйеге қосылады, ал сервер жағы (Backend) барлық бизнес-процестерді басқарады, деректер қорымен жұмыс істейді және өтініштерді қауіпсіз жүзеге асырады. PostgreSQL базасы жұмыс іздеушілердің түйіндемелерін, вакансиялар ақпаратын және платформаның басқа да деректерін қауіпсіз сақтайды.

2. ПРАКТИКАЛЫҚ БӨЛІМ ЖӘНЕ СЕРВЕРГЕ ОРНАЛАСТЫРУ

2.1 Орындалған функционалдар және платформа интерфейсі
Платформада келесі негізгі функционалдар толықтай іске асырылды:
- Рөлдік басқару: Жұмыс іздеуші (USER), Жұмыс беруші (EMPLOYER) және Әкімші (ADMIN) рөлдері.
- Аутентификация: Қолданушыларды тіркеу және жүйеге кіру (JWT токендер арқылы).
- CRUD операциялары: Вакансияларды қосу, өңдеу, өшіру. Резюме жасау.
- Іздеу және Фильтрлеу: Жұмыстарды категориялары бойынша іздеу.
- Өтінім қалдыру: Вакансияларға өтінім (application) жіберу.
- PWA (Progressive Web App): Сайтты ұялы телефонға және компьютерге қосымша (приложение) ретінде орнату мүмкіндігі.

Скриншоттар:
(Осы жерге өзіңіздің сайтыңыздан 3-4 скриншот қойыңыз: Басты бет, Вакансиялар беті, Тіркелу терезесі)

2.2 Жобаны серверге орналастыру (Deploy)
Жоба толығымен бұлтты серверлерге орналастырылды және ғаламтор желісінде қолжетімді.
- Сервер атауы: Vercel (Frontend үшін) және Render.com (Backend және Database үшін).
- Жоба сілтемесі: https://mansap-app.vercel.app/
- Git репозиторий сілтемесі: https://github.com/BaiganMarzhan008/mansap-app.git

2.3 Тестілеу нәтижелері және кездескен қателерді шешу
Тестілеу нәтижесі:
- Функционалдық тестілеу: Тіркелу, кіру және вакансия қосу процестері 100% қатесіз жұмыс істеді.
- API тестілеу: Барлық REST API маршруттары тексеріліп, сервермен сәтті байланыс орнатылды.
- PWA тестілеу: Chrome және Safari браузерлері арқылы «Add to Home Screen» функциясы сәтті іске асты.

Кездескен қателер және шешу жолдары:
1. Қате: Жергілікті (локальный) PostgreSQL базасына қосылу кезінде «The server does not support SSL connections» қатесі шықты.
   Шешімі: `backend/config/database.js` файлындағы `dialectOptions.ssl` баптауын алып тастау арқылы шешілді.
2. Қате: Терминалдан GitHub-қа жүктеу кезінде `fatal: not a git repository` қатесі шықты.
   Шешімі: Команда қате папкада орындалғаны анықталып, негізгі папкаға өтіп, жаңа `git remote add origin` жасалды.
3. Қате: Сайтты PWA ретінде орнату батырмасы бірден шықпады.
   Шешімі: `vite.config.js` ішіндегі PWA баптауларында иконкаларға `purpose: "any maskable"` қасиеті қосылды.

ҚОРЫТЫНДЫ

Қорытындылай келе, аталған курстық жұмыс барысында қойылған мақсат толықтай жүзеге асырылды. Жұмыс нәтижесінде жұмыс іздеушілер мен жұмыс берушілер арасындағы байланысты тиімді ұйымдастыруға арналған заманауи «Mansap App» веб-платформасы әзірленді.
Жоба аясында веб-платформаның құрылымы жобаланып, негізгі функционалдар енгізілді. Локальді ортада жасалған қосымша толығымен бұлтты серверлерге (Vercel және Render) көшіріліп, интернетте нақты жұмыс істейтін өнімге (Production) айналды. Сонымен қатар, қауіпсіздік шаралары сақталып, заманауи PWA технологиясы енгізілді. Жоба барлық қойылған талаптарға толық жауап береді және болашақта одан әрі дамытуға дайын.
